Инструкция запуска сервиса для linux:

1. Открыть папку my_flask_app в терминале
2. Создать виртуальную среду (например, python3 -m venv myenv) или подключиться к существующей
3. Активировать виртуальную среду (для созданной выше среды: source myenv/bin/activate)
4. Установить требуемые библиотеки (pip install -r requirements.txt)
5. Запустить сервис командй flask run
